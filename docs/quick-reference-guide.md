## Technology Stack Summary

### Frontend
- **Framework:** React 18 with Vite
- **UI Library:** Shadcn UI
- **Styling:** Tailwind CSS
- **State Management:** Context API
- **HTTP Client:** Axios
- **Routing:** React Router v6

### Backend
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB with Mongoose ODM
- **Authentication:** JWT + bcrypt
- **Validation:** Joi
- **Email:** Nodemailer

### External Services
- **Storage:** Google Drive API
- **Collaboration:** Google Docs API
- **Deployment:** Vercel (Frontend), Render (Backend), MongoDB Atlas M0

### Development Tools
- **Version Control:** Git + GitHub
- **API Testing:** Postman/Insomnia
- **Database Client:** MongoDB Compass
- **Code Editor:** VS Code
- **Design:** Figma

## Risk Management

### High Risks
| Risk | Probability | Impact | Mitigation |
|------|-------------|--------|------------|
| Google Drive API limits exceeded | Medium | High | Monitor usage, implement caching |
| 512MB RAM limit reached | Medium | High | Optimize code, stream files |
| UAT participants unavailable | Low | High | Schedule early, have backup participants |
| Delays in adviser approval | Medium | Medium | Regular check-ins, advance submissions |

### Medium Risks
| Risk | Probability | Impact | Mitigation |
|------|-------------|--------|------------|
| Free tier service limitations | Medium | Medium | Plan for upgrade if needed |
| Integration issues with Google APIs | Medium | Medium | Early testing, fallback plans |
| Deployment issues | Low | Medium | Test deployment early, have rollback plan |

### Low Risks
| Risk | Probability | Impact | Mitigation |
|------|-------------|--------|------------|
| Browser compatibility issues | Low | Low | Cross-browser testing |
| Minor bugs in UAT | High | Low | Bug tracking and prioritization |
